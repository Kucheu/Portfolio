      <h3>Steam inventory value calculator (2026)</h3>
      Automating the process of calculating the value of steam equipment. Creating a graph of the change in equipment value over time</br>
      </br>
      The program was written in Python and uses:</br>
      1. Steam API</br>
      2. urllib to retrieve data from the API, and manage errors if necessary</br>
      3. json to decode api result and store items value</br>
      4. matplotlib to create a plot from the collected data</br>
      5. the application also stores item values ​​for a given day, so as not to exceed the daily limit of requests to the Steam API</br>
            </br></br>
